国外中小学教师人事关系的调整方式及其影响

在讨论国外中小学教师人事制度时，研究者多使用公务员、教育公务员、公务雇员等概念描述国外教师的法律地位，这些概念将一国的教师人事关系法律制度简单化了，不能呈现国外中小学教师人事关系法制化的基本状况，也会得出一些不恰当的结论。本文认为法律地位并不能确定中小学教师具体的权利与义务，其只是指示教师人事关系调整方式的不同。教师与用人单位之间的各项权利义务关系内容非常繁多，涉及招聘、工资待遇、社会保险、工作时间和环境、考核评价、培训、职业发展等事项，在确定教师与学校或政府在上述事项上的关系时，各国采取了两种不同的调整方式。一种是以公法来调整的教师人事关系，包括适用公务员法或者专门针对中小学教师的立法，教师与用人主体的关系由法律法规确定；另一种是以劳动法为调整方式，教师与用人主体以劳动合同确定双方的权利义务关系，劳动法对劳动合同的内容做种种限定，有些国家制定了针对公职人员或教师的特殊劳动合同法，有些国家适用通用的劳动合同法。劳动合同包括教师个体与用人主体之间的个体合同、教师集体与用人主体签订的集体合同，集体合同还可以分全国性的集体合同和区域范围内的集体合同。就单个国家而言，一般采用上述调整方式的一种或以一种为主兼用另一种方式。使用一般公务员法或特殊公务员法调整中小学教师人事关系的国家，比如法国、德国、西班牙、荷兰、土耳其等，使用劳动法调整的国家，比如瑞典、丹麦、巴西、英国、美国等。为了分析国外中小学教师人事制度具体情况以及教师人事制度的效果，本文根据使用人事关系调整方式的主导模式将国家或地区分为两类：以公法调整为主和以劳动法调整为主。在分析的68个国家或地区当中，以公法为主调整教师人事关系的有29个，以劳动法为主调整的有39个，通过分析这些国家或地区的教师人事制度，可以得出以下结论。1

教师人事关系的调整方式与教师的具体权利义务设定没有必然联系

仔细分析各国教师人事关系法制的具体内容，可以发现无论采用公法为主的国家，还是以劳动法为主的国家，其只是调整人事关系的法律适用不同，其与中小学教师不同事项上的具体权利义务配置并不存在必然联系。2一方面，使用同样调整方式的国家之间，在中小学教师具体的权利义务设定方面存在很大差异。比如，教师职业结构涉及教师的类别与级别和与之相应的工资等级，同样是公法调整为主的法国与德国，法国教师的职业结构比较复杂，而德国教师之间的差异则并不大。再比如，同样是以劳动法调整为主的国家，教师的雇用主体也不一样，有些国家学校是雇主，比如捷克，有些国家地方政府是雇主，比如瑞典与丹麦。另一方面，使用不同调整方式的国家之间，在中小学教师具体权利义务设定上存在很多相似之处。比如，以公法调整为主的国家，教师人事关系多数具有终身保障的性质，但采用劳动合同制的国家基本上都有无终止期限合同。另外，即使在一些以公法调整为主的国家，也存在考核期，比如，实行公务员制度的德国，教师成为终身公务员之前有2.5或3年的见习期；在劳动合同制的国家，在获得无终止期限合同之前，也有一段固定期限合同。在教师招聘方面，以公法调整为主的荷兰和挪威，以劳动法调整为主的瑞典和丹麦都是由学校直接招聘。再比如，无论在公法调整为主还是劳动法调整为主的国家，教师工会在教师权利义务设定中都起到了至关重要的作用。在公务员制的国家，比如，西班牙教师立法过程中专门有与教师工会协商的环节，荷兰公立学校教师工资与工作时间主要由教师工会与地方政府之间的集体协议确定。在以劳动法调整为主的国家中，瑞典教师的工资待遇与工作条件由地方政府联合体与教师工会协商确定，丹麦是由地方政府与教师工会确定教师与政府之间的协议。

教师人事关系的调整方式与教师权利保障并不存在必然联系

为了评估不同调整方式的选择与教师利益保障之间的关系，本文从教师工资收入、工作时间和职业稳定性三个方面，来分析中小学教师人事关系调整方式与教师权利保障之间的关系。使用本文对部分国家的教师人事关系调整方式的统计，再使用经济合作与发展组织（OECD） 统计的28个国家中学教师平均工资信息和2018年的教师教学国际调查(TALIS)数据中44个国家或地区中学教师工作时间和人事关系期限的数据，来分析教师人事关系调整方式与教师工资待遇、工作时间和职业稳定性的关系。

首先，使用OECD统计的28个国家的教师相对平均工资信息，该指标是以国家或地区的全职公办中学教师的平均工资除以该国或地区高等教育毕业生的平均工资计算而得，该指数一定程度上可以反映各国中学教师在本国的相对收入水平。图1反映的是教师人事关系调整方式与教师相对平均工资的关系图，由图可知，各国教师人事关系调整方式与教师收入水平并不存在相关性。中小学教师人事关系以公法调整为主的国家，其教师收入水平并不一定高，以劳动法为主调整的国家的教师收入水平也并不一定低。比如，同样是公法调整为主的国家，德国和葡萄牙的教师收入水平比较高，而挪威和匈牙利的教师收入水平则较低。同样是以劳动法调整为主的国家，哥斯达黎加和立陶宛的教师收入水平比较高，而捷克和美国的教师收入水平则较低。

图2A反映的是教师人事关系调整方式与教师工作时间之间的关系。教师工作时间是根据TALIS调查中由各国家或地区的教师报告的周工作时间的统计平均值。由图可知，各国教师人事关系调整方式与教师工作时间没有绝对关系。以公法调整为主的国家，日本和哈萨克斯坦教师的工作时间超过50小时，而沙特阿拉伯和土耳其的教师工作时间只有29和32小时。以劳动法调整为主的国家，英格兰和加拿大教师的工作时间为52和48小时，而格鲁吉亚和意大利为28和31小时。

# 合同数据预处理
contract_legal <- teacher %>%
  group_by(idcntry, tt3g09) %>%
  summarise(n = n()) %>%
  ungroup() %>%
  group_by(idcntry) %>%
  mutate(
    pro = n / sum(n),
    perpro = ifelse(tt3g09 == 1, pro, 0)
  ) %>%
  mutate(employ = as.character(tt3g09)) %>%
  left_join(auto, by = c("idcntry" = "country2018"))

contract_indefinite <- contract_legal %>%
  filter(tt3g09 == 1) %>%
  select(country_chn, pro, contract)
# 生成长期合同占比柱图
fig_contract <- contract_indefinite %>%
  ggplot(aes(y = fct_reorder(country_chn, pro), x = pro, fill = as.character(contract))) +
  geom_col(show.legend = T) +
  geom_text(aes(label = round(pro * 100, 0), hjust = 1.2),
    color = "white", size = 3
  ) +
  labs(x = "教师占比", y = NULL) +
  scale_fill_manual(
    name = "调整方式", values = c("grey20", "grey60"),
    labels = c("公法", "劳动法")
  ) +
  scale_x_continuous(
    breaks = seq(0, 1, 0.2), labels = label_percent(),
    expand = expansion(mult = c(0, 0.01))
  ) +
  theme_minimal_vgrid(font_size = 10) +
  theme(legend.position = "bottom", text = element_text(family = "FangSong")) +
  guides(fill = guide_legend(nrow = 1))

# 合并工作时间图和长期合同图
library(patchwork)
fig_hour + fig_contract +
