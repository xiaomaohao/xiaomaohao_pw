教育政治学

课程简介

政治，作为一种理念、制度、活动，无时不刻影响着教育内容的选择、教育资源的分配、教育管理的运行。教育，作为一种主动对个体认知、价值和技能施加影响的活动，无时不刻影响着个体政治观念、政治参与意识和参与能力的形成。缺乏政治学的视角研究教育问题，有时看不到教育问题的本质，是残缺的；如果不能理解教育在政治文明发展过程的重要意义，政治也将是不稳定的。教育政治学即是探究教育与政治复杂关系的学科。

课程目标

  1. 使学生了解不同层级政府、政府内部各部门及各利益团体在教育中的作用与影响，为学生理解教育立法、教育政策制定以及教育行政管理实践提供基础；
  2. 使学生掌握以政治学的视角来分析教育现象与教育问题的能力。

课程内容

  1. 我国的政治体制及教育管理体制。主要分析我国政治体制的特征以及基础教育和高等教育管理体制；
  2. 教育政策制定过程分析。以《民办教育促进法》制定过程为例，分析立法、行政、民众等不同主体在政策制度过程中的作用，并就政策制定过程的本质展开讨论；
  3. 教育财政权力的分配。以义务教育为例，分析政府是否有义务承担保障教育经费的责任？哪级政府有义务承担教育经费的保障责任？如何落实政府义务教育经费保障责任？上级政府的财政经费如何分配？
  4. 政府教育供给与民众教育需求关系分析。以中等职业教育”普职大体相当”和”免补政策”为例，探讨政府教育供给与民众教育需求之间的矛盾。
  5. 教育人事权的分配问题。以中小学教师编制管理为例，分析编制的属性，编制权是不同层级政府、政府不同部门的分配问题，编制在不同部门的分配问题。
  6. 教育的政治功能分析。分析教育与公民参与意识、参与能力，教育与民主化进程，教育与行政质量，教育与社会稳定之间的关系。

课程要求

  1. 课程以专题讨论的形式进行，学生须阅读相关材料；
  2. 课堂上学生需积极参与讨论发言；
  3. 每位学生要作一次专题报告；
  4. 完成一篇不少于5000字的与本课程主题相关的论文；

考核方式

  • 出勤：10%
  • 课堂表现：20%
  • 专题报告：20%
  • 结课论文：50%
