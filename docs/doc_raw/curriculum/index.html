开设课程

本人开设的课程包括研究生专业课教育法律实务、教育政治学，方法课数据科学与教育、量化研究论文写作指导，本科生课程教育管理纠纷案件分析等。

专业课

教育法律实务

　　法律的生命在于实施，教育法亦不例外。我国教育法律体系正在逐步形成，如何将其运用于教育实践，有效处理教育领域中不同主体之间的权利义务纠纷，是判断教育法治功能是否实现的重要标准。教育法制的完善亦需要个案中形成的规则、标准反哺教育法律的制定。本课程以学生、教师、学校和行政部门等主体展开，展现教育领域形形色色的纠纷案例，尝试使用法律分析和解决这些纠纷，帮助学生形成正确的法治理念和法治思维。

教育政治学

　　政治，作为一种理念、制度、活动，无时不刻影响着教育内容的选择、教育资源的分配、教育管理的运行。教育，作为一种主动对个体认知、价值和技能施加影响的活动，无时不刻在影响着个体政治观念、政治参与意识和参与能力的形成。缺乏政治学的视角研究教育问题，有时看不到教育问题的本质，是有残缺的；如果不能理解教育在政治文明发展过程的重要意义，政治也将是不稳定的。教育政治学即是探究教育与政治复杂关系的学科。

方法课

数据科学与教育

　　数据科学涉及对数据清理、统计和分析以获得知识和洞见的理论和技术。本门课程主要是帮助学生掌握数据分析的基本步骤、方法、原理和规范。课程依赖于但不同于传统的概率与统计知识，更侧重对数据挖掘技术的训练，课程内容包括数据清理、探索性分析技术（图表制作）和一般性的相关性分析。课程使用的统计分析软件为R。

量化研究论文写作指导

　　本课程是一门量化论文写作实践指导课，学生将在老师指导下完成论文从选题到成稿的完整过程，课堂以学生习作为研讨对象，发现论文写作中的具体问题，在实践中改进。课程将依次围绕论文选题、文献综述、理论框架搭建、研究设计、数据分析和结果呈现等展开，以求课程结束，学生完成一篇论文的完整写作过程。课程强调同伴学习，除教师的点评之外，鼓励学生互相点评和提建议。
