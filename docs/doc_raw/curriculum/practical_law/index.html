教育法律实务

课程简介

学生可不可以起诉学校？在校生可否签订劳动合同？买了学区房无法兑现怎么办？应聘教师遭到歧视怎么办？学校索取违约金是否合法？教师工伤如何认定？校名简称冲突如何解决？违法办学问题日益突出，谁的责任？校园暴力与校园欺凌问题如何解决？等等。本课程将围绕以上问题展开，结合最新最富有争议的案例进行分析，以案说法，使学生熟悉教育相关法律法规；针锋论辩，培养学生运用法律思维解决教育问题的能力。

课程目标

  1. 了解教育领域的相关法律；
  2. 了解教育法律实务的常见纠纷案例；
  3. 掌握运用法律解决教育管理纠纷的能力；
  4. 培养基本的教育法治理念与法治思维；

课程内容

  1. 课程内容简介       内容简介，法律及教育法基本知识；
  2. 学生权益专题一     学位撤销纠纷案例分析；
  3. 学生权益专题二     开除学生纠纷案例；
  4. 学生权益专题三     受教育权纠纷之一：冒名顶替上学纠纷；学区房纠纷；
  5. 学生权益专题四     受教育权纠纷之二：最近入学权纠纷；学生就业权纠纷；
  6. 学生权益专题五     学生转专业、转学，参与学校管理，公正评价权等纠纷；
  7. 教师权益专题一     教师职业相关制度简介；
  8. 教师权益专题二     教师招聘纠纷；
  9. 教师权益专题三     教师辞职、辞退、离职、解除用人关系等纠纷；
  10. 教师权益专题四    社会保险待遇纠纷，如工伤认定，退休待遇等；
  11. 教师权益专题五    教师工资福利相关纠纷；
  12. 教师权益专题六    教师在职权利纠纷，如教案著作权，惩戒学生权等；
  13. 学校权益专题一    校园人身伤害事故纠纷；
  14. 学校权益专题二     校名冲突纠纷，博士学位点审批纠纷；
  15. 学校权益专题三    民办学校相关纠纷；
  16. 行政主体专题一    教育行政执法纠纷，涉及审批、处罚、信息公开等；

授课形式

课程以案例教学为主，重在讨论，教师讲授为辅。

阅读材料

阅读材料主要研讨案例的司法判决文书、媒体相关报道和一些期刊文章。

考核方式

  • 出勤：10%
  • 作业：20%
  • 课堂表现：30%
  • 结课论文：40%
